# Legacy Quantlets

These Quantlets are from the original paper:

> Pele, D.T., Lessmann, S., Hardle, W.K. (2025). In the Beginning was the Word: LLM-VaR and LLM-ES. *Expert Systems with Applications*, 295, 128676.

They used GPT-3.5/4/4o on 9 assets with rolling calibration windows (w = 30, 45, 60, 90, 120, 150). The current paper replaces these with 5 open-weight TSFMs on 24 assets with a single 70/30 calibration split.

Preserved here for reproducibility of the original results.
