# CFP_Simulation_Study

Monte Carlo simulation under controlled misspecification.

## DGPs
1. Normal (correctly specified)
2. Student-t(5) (mild)
3. Student-t(3) (moderate)
4. Skewed-t(3, -0.5) (heavy + asymmetric)
5. Mixture 0.95*N(0,1) + 0.05*N(0,5) (jump component)
